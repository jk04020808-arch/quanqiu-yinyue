Global Beats 3D

3D Music Trends · 交互式全球音乐数据可视化

Step 1 / Intro

沉浸式 3D 探索

打开网页，映入眼帘的是一个平滑自转的 3D 地球。半透明的玻璃态引导层温和地提示用户：「拖拽旋转，滚动缩放，点击探索」。4秒后自动淡出，将舞台完全留给数据与音乐。

探索全球音乐
拖拽旋转 · 滚动缩放 · 点击探索

Step 2 / Data

国家音乐排行榜

点击任意国家（如日本），地球停止自转并聚焦。右侧毛玻璃面板平滑滑入，呈现基于真实数据的 Top 5 热门歌曲、流媒体播放量柱状图以及音乐流派分布环形图。数据源自 iTunes 33 国与网易云音乐实时打通。

Step 3 / Heatmap

全球流行热力图

在面板中点击某首歌曲，D3.js 驱动的颜色插值引擎瞬间将地球切换为热力图模式。紫-青渐变直观反映该歌曲在各国的排名分布。顶部 HUD 实时显示状态，左下角图例辅助数据解读。

正在显示《Shape of You》热力分布

Step 4 / Audio

30 秒高潮试听

悬停歌曲封面点击播放，系统自动通过 iTunes Search API 检索该歌曲的 30 秒高潮音频。底部弹出精美的播放控件，伴随跳动的绿色音柱动画和 Toast 提示，带来所见即所听的闭环体验。

Step 5 / Timeline

27 年时间穿越

底部精简的时间滑块覆盖了 2000 年至 2026 年按月粒度的历史数据。拖动滑块，地球材质与排行榜数据实时重绘，带你回顾过去近三十年间全球流行音乐的演变轨迹。

Step 6 / Search

全局实时搜索

内置全局搜索功能，输入歌手或歌曲名称，通过后端 Express 聚合接口实时检索。搜索结果支持一键播放试听，或直接在 3D 地球上生成该特定歌曲的全球热度图谱。

Step 7 / Drill-down

中国省级数据钻取

点击中国版图，通过「进入省级视图」按钮，镜头平滑拉近，GeoJSON 多边形从国家级无缝切换至中国 34 个省级行政区。揭示从北京到四川差异化的地域音乐偏好。

Step 8 / Shortcuts

极客快捷键支持

为提升桌面端体验，系统内置完善的键盘映射：按下 ESC 键可根据当前层级依次退出热力图、关闭侧边栏、返回全球视图；空格键全局控制音频的播放与暂停。

项目核心技术特点

📊

真实数据驱动

非 Mock 数据。深度聚合网易云音乐 API (4 大官方榜单) 及 iTunes RSS API，覆盖 33 个核心国家与地区。

🌐

3D WebGL 引擎

基于 react-globe.gl 深度定制，自定义 GeoJSON 多边形着色、拔高算法与大气层散射光照模型。

🗄️

SQLite WAL 模式

采用 better-sqlite3 构建零配置轻量级数据库，单文件 85MB 支撑 380,000+ 条高维历史排行数据极速查询。

🎨

动态热力着色

集成 D3.js 颜色比例尺插值算法，将抽象排名数据实时转换为 3D 空间中的视觉热力梯度图。
